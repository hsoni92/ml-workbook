As we have already studied in the previous video, in deep networks, not all parameters behave the same way. Some receive very large gradients, others receive very small or infrequent gradients. If we use single learning rate for all the parameters, two things can happen.
Parameters with large gradients can overshoot and destabilize the training. And parameters with small gradients barely move and learn extremely slowly. SGD has no mechanism to handle this imbalance. RMS prop was introduced specifically to address this issue.
The core idea of RMS prop is simple but powerful. It keeps track of how large the gradients have been recently for each parameter. If the parameter consistently sees large gradients, RMS prop automatically reduces its learning rate or the step size. If the parameter sees small gradients, RMS prop increases its effective learning rate. So, instead of one global learning rate, each
each parameter gets its own dynamically adjusted step size. This allows RMS prop to balance learning across the parameters, dampen the unstable directions. Since it changes for each parameter, it can boost the slow learning direction specifically and even smooth the updates even when the gradients are noisy or sparse.
