Once an MLP is defined and trained, every prediction it makes is produced by a forward pass. A forward pass simply means executing the stored computations of the network on a single input.
Let us now trace what happens when one input vector enters the network. Suppose, we receive an input vector x. This vector is directly fed into the first hidden layer of the network. At this stage, the network is not making any decision. It is only beginning a sequence of transformations. The same input is not reused at every layer. Instead, each layer works on a transformed version.

This continues layer by layer until the signal reaches the output layer. An important conceptual point is this. The original input vector is never used again directly. Each layer only sees the transformed representation produced by the previous layer.
The final step of the forward pass happens at the output layer. The output layer converts the last hidden activation vector into a prediction that is meaningful for the task. For binary classification, it typically produces a probability. For multiclass classification, it produces class scores or probabilities. For regression, it produces a real valued output.
At this point, the network has completed its forward pass and generated its prediction. The forward pass is the core computation of a neural network. Every prediction in real systems, whether in image recognition, recommendation systems, or medical diagnosis, is produced by a forward pass. Training algorithms will repeatedly use the same forward computation and then adjust the weights based on the observed errors.
