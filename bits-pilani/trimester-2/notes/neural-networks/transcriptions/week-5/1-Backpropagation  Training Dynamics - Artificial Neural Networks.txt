So far in this course, you have learned how neural networks are structured and how they compute outputs using a forward pass. You now understand MLPs, hidden layers and activation functions.
But one fundamental question has not yet been answered. How does a neural network improve itself using data?
In other words, when a network makes a mistake, how does it know what went wrong and what exactly needs to be changed inside the network?
This question is the central focus of this entire module.
The whole idea behind training a neural network model is to adjust its internal parameters, i.e. weights and biases, so that the predictions become better.
Training involves the following steps.
We start with the forward pass, which we already know now.
In forward pass, we compute the predictions from the model.
Next step is loss computation, which involves comparing the predictions with the reality.
This tells us about the error that the model is making.
After this, we compute the gradients of the parameters to determine how much each parameter is contributing to the error.
And then we correct or improve the parameters gradually.
This mathematical process that makes this possible is called backpropagation.
Backpropagation tells us how responsibility for error is distributed across the entire network.
