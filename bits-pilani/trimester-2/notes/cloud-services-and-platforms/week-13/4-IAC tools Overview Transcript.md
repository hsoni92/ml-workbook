# IaC Tools Overview

## Terraform
- Multi-cloud declarative IaC using providers.
- Uses HCL configuration and state file.
- Strong module ecosystem and plan/apply workflow.

## Cloud-native tools
- AWS CloudFormation models AWS resources in templates.
- AWS CDK lets developers define infrastructure using programming languages that synthesize templates.
- Azure Bicep/ARM and Google Deployment Manager serve similar provider-native roles.

## Configuration tools
- Ansible, Chef, and Puppet focus more on software/configuration management.
- They can provision infrastructure too, but Terraform is commonly preferred for cloud resource lifecycle.
- Exam questions usually distinguish provisioning from configuration.
