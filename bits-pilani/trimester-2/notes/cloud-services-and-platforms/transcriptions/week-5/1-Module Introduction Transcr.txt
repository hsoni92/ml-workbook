website, API, database, or any background job needs a machine to execute the code.
In AWS, that machine is usually an EC2 instance.
This module focusses on understanding the EC2 instance and the idea of
virtualisation. So each of these virtual servers feels like its own computer with its own

is like renting a computer from Amazon's data centre.
You choose how powerful it should be, what operating system it runs, and how secure
it is, and when it should be turned on or off. Most importantly, you pay only for the time

scaling and traffic.
