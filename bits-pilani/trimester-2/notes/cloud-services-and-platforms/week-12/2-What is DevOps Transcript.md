# What Is DevOps

## Definition
- DevOps is an operating model where development and operations share responsibility for delivery and reliability.
- It emphasizes automation, collaboration, small changes, observability, and continuous improvement.
- It reduces handoff delays between writing code and running code.

## Core practices
- Version control for code and infrastructure.
- Automated build/test/deploy pipelines.
- Monitoring and feedback from production.
- Blameless incident learning and iterative improvement.

## Cloud connection
- Cloud APIs allow infrastructure to be provisioned by pipeline.
- Managed services reduce operational toil.
- IAM and policy controls keep automated delivery governed.
