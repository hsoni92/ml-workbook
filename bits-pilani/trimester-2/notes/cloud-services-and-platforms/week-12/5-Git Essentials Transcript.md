# Git Essentials

## Daily workflow
- Clone repository to local machine.
- Create branch for change.
- Stage and commit changes.
- Push branch to remote.
- Open pull request and merge after review/checks.

## Commands to know
- git status shows working tree state.
- git add stages files.
- git commit records snapshot.
- git push uploads commits.
- git pull/fetch gets remote changes.

## Exam and practice traps
- Commit small logical changes with meaningful messages.
- Avoid committing secrets, generated junk, or local environment files.
- Resolve merge conflicts by understanding both sides, not blindly choosing one.
