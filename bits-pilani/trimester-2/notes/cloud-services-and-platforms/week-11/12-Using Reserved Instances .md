# Using Reserved Instances

## What Reserved Instances do
- Provide discount for committing to usage over 1 or 3 years.
- Best for steady, predictable workloads.
- Commonly compared with On-Demand, Spot, and Savings Plans.

## Types and choices
- Standard RI gives larger discount but less flexibility.
- Convertible RI allows exchange for different attributes with lower discount.
- Regional vs zonal scope affects capacity reservation behavior.

## Exam decision
- Use Reserved/Savings commitments only after usage baseline is known.
- Use On-Demand for unpredictable short-term needs.
- Use Spot for fault-tolerant interruptible workloads.
