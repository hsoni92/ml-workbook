# AWS Budgets and Cost Explorer

## AWS Budgets
- Sets planned cost/usage/reservation thresholds.
- Can notify when actual or forecasted spend crosses threshold.
- Useful for guardrails by account, service, tag, or linked account.

## Cost Explorer
- Analyzes historical spend and usage trends.
- Groups by service, linked account, region, tag, usage type, or purchase option.
- Helps identify rightsizing and savings opportunities.

## Exam distinction
- Budgets alerts you; Cost Explorer investigates patterns.
- Budget does not automatically reduce architecture cost unless actions/policies are configured.
- Tags make both tools much more useful.
