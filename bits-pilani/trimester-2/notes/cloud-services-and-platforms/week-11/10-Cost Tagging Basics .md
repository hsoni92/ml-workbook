# Cost Tagging Basics

## Why tags matter
- Tags attach business context to resources.
- Cost allocation tags let bills be grouped by team, product, environment, or owner.
- Without tags, shared cloud spend becomes guesswork.

## Tag design
- Common keys: Owner, Project, Environment, CostCenter, Application, DataClassification.
- Use consistent spelling and allowed values.
- Activate cost allocation tags in billing tools for reporting.

## Governance
- Use tag policies/SCPs/Config rules where appropriate to enforce standards.
- Automate tagging in IaC templates.
- Tagging prevents confusion but does not replace rightsizing or budgets.
