# Types of Cloud Storage

## Object storage
- Stores data as objects with metadata and key names.
- Best for unstructured data, images, logs, backups, datasets, and static web content.
- S3 is the primary AWS example.

## Block storage
- Presents raw block devices to compute instances.
- Best for OS boot disks, databases, and applications requiring low-latency filesystem control.
- EBS is the primary AWS example.

## File storage
- Provides shared hierarchical filesystem access.
- Best for multiple servers needing common files, content repositories, or shared ML preprocessing space.
- EFS is the primary AWS example.
