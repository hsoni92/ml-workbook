# Cloud in Real-World Applications

## Objective
Connect cloud fundamentals to practical use in AI/ML, IoT, and web/mobile ecosystems.

## 1) Cloud in AI and Machine Learning

### Why Cloud Is Essential
- AI training workloads need large-scale compute (GPU/TPU).
- On-prem high-performance infrastructure is costly and difficult to maintain.
- Cloud enables rapid experimentation and parallel model training.

### Common Cloud AI Capabilities
- Managed ML platforms
- Distributed training environments
- Pre-built AI APIs (vision, speech, NLP)
- Scalable model deployment endpoints

### Business Impact
- Reduces entry barrier for AI adoption
- Improves training time and iteration speed
- Enables smaller teams to build production AI systems

## 2) Cloud in IoT (Internet of Things)

### Why Cloud Is Essential
- IoT ecosystems generate continuous data streams from large device fleets.
- Centralized ingest, storage, and analytics are required for operational value.

### Typical Cloud Functions in IoT
- Device registration and management
- Streaming data ingestion
- Event processing and automation
- Secure access and monitoring
- Long-term analytics pipelines

### Example
Smart-city systems collect data from lights, cameras, water meters, and transport sensors to improve urban operations.

## 3) Cloud for Web and Mobile Applications

### Common Cloud-Backed Components
- Application hosting
- User authentication
- Databases and storage
- Media delivery and CDN
- Notifications and messaging
- Live streaming and real-time APIs

### Elastic Behavior Example
Food delivery traffic spikes during lunch/dinner and scales down late night, balancing user experience and cost.

## Conceptual Summary
Cloud is not a background utility only; it is a core enabler for:
- Intelligence (AI/ML)
- Connected ecosystems (IoT)
- Global digital products (web/mobile)

## Exam-Focused Questions
- Why is cloud preferable to local infrastructure for AI training?
- What cloud functions are mandatory in large IoT systems?
- Explain elasticity in consumer app traffic patterns.

## Quick Revision
- AI/ML: cloud provides compute acceleration and managed tooling.
- IoT: cloud provides scale for ingest, processing, and analytics.
- Web/mobile: cloud provides reliable, elastic, globally reachable backends.
