# Case Studies: Netflix, Uber, Dropbox

## Objective
Understand cloud value through three high-impact business cases with different workload profiles.

## 1) Netflix: Global Streaming at Massive Scale

### Cloud Challenges Solved
- Large, unpredictable traffic surges
- Global content delivery requirements
- High availability expectations
- Storage and delivery of large media catalogs

### Why Cloud Was Critical
- Elastic scaling for peak events
- CDN-based global distribution
- Fault tolerance to reduce interruption risk
- Analytics support for personalized recommendations

### Key Lesson
Cloud enables media services to operate with global reach and demand volatility.

## 2) Uber: Real-Time, Latency-Sensitive Operations

### Cloud Challenges Solved
- Continuous rider-driver matching
- Geolocation and ETA calculations
- Dynamic pricing updates
- Real-time notifications and messaging
- Transaction processing at city/global scale

### Why Cloud Was Critical
- Low-latency infrastructure for real-time decisions
- Rapid scaling during peak commuting windows
- Geographic expansion support through distributed infrastructure

### Key Lesson
Cloud is essential for systems requiring continuous, real-time coordination.

## 3) Dropbox: Global File Sync and Collaboration

### Cloud Challenges Solved
- Multi-device file synchronization
- Availability and durability for user data
- Secure sharing and collaboration
- Redundancy and failover requirements

### Why Cloud Was Critical
- Reliable object/file storage at scale
- Fast sync propagation across devices and regions
- Identity and access workflows integrated with collaboration features

### Key Lesson
Cloud powers persistent, always-available collaboration experiences.

## Comparative Insight
| Company | Primary Cloud Strength |
|---|---|
| Netflix | Scalability + global distribution |
| Uber | Real-time processing + low latency |
| Dropbox | Storage reliability + synchronization |

## Exam-Focused Takeaways
- Use these cases to justify cloud adoption drivers with evidence.
- Be able to map each company to specific cloud characteristics.
- Explain why these services would be difficult to run effectively without cloud infrastructure.

## Quick Revision
- Netflix = scale and distribution.
- Uber = real-time orchestration.
- Dropbox = durable sync and collaboration.
