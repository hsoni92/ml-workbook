# Apex Project-2 (Week 2)

ANN-based customer response prediction outline (P4) using the UCI Bank Marketing dataset.

## Setup

Requires [uv](https://docs.astral.sh/uv/).

```bash
uv sync          # create .venv and install dependencies
make data        # download bank-full.csv into data/
```

## Run notebook (no browser)

Executes all cells and writes `Apex2_Week2_Notebook_Template1_executed.ipynb`:

```bash
make
# or
make execute
```
