## Lesson 2: Seven Stages of Data Analysis

### APFMRRI - Seven Stages
1. **A**cquire: Gathering data from its source
2. **P**arse: Structuring data to define meaningful components
3. **F**ilter: Eliminating irrelevant details to highlight key data
4. **M**ine: Analyzing data to reveal underlying trends and insights
5. **R**epresent: Beginning with a simple visual to identify complex relationships
6. **R**efine: Enhancing visual representation to transform data into clear information
7. **I**nteract: Sharing analysis with stakeholders, enabling exploration

### Types of Charts (CDCR)
- **C**omparison Charts
- **D**istribution Charts
- **C**omposition Charts
- **R**elationship Charts

## Lesson 10: Effective Visuals Design

### Main Categories of Clutter (FGAOL)
1. **F**rame: Background and borders
2. **G**raphical objects: Bars, lines, or points
3. **A**xes: Reference lines for measurement
4. **O**verlay objects: Explanatory elements
5. **L**abels: All text elements

### Types of Memory (ISL)
- **I**conic Memory: Ultra-fast, unconscious visual sensory memory
- **S**hort-Term Memory: Conscious "working" memory
- **L**ong-Term Memory: Stores patterns, concepts, and experiences

### Pre-attentive Attributes Categories (FCPM)
- **F**orm: Orientation, shape, line length, line width, size, curvature
- **C**olor: Hue and intensity (saturation)
- **P**osition: Spatial position, enclosure, added marks
- **M**otion

## Lesson 11: Designing Visuals

### Data Types (NOIR)
- **N**ominal Data: Qualitative categories (Gender, Blood Group)
- **O**rdinal Data: Ordered set (Customer Reviews)
- **I**nterval Data: Fixed interval sets (IQ level, temperature)
- **R**atio Data: All arithmetic operations meaningful (Revenue, Sales)

### Three Visual Operations (DAE)
1. **D**etection: Recognizing patterns or values
2. **A**ssembly: Grouping geometric objects to understand relationships
3. **E**stimation: Assessing magnitude or comparisons

### Visual Encoding Effectiveness (PPLAVCS)
- **P**osition on Common Scale: Most effective
- **P**osition on Identical but Non-Aligned Scales: Effective
- **L**ength: Very effective
- **A**ngle: Useful for ranking or estimation
- **V**olume / Density / Saturation: Useful but less effective
- **C**olor: Best for discrimination, not ranking
